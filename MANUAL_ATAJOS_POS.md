# 🚀 Manual de Atajos de Teclado - NEXUM POS

Este manual detalla las teclas de acceso rápido diseñadas para agilizar la operación en caja. El uso de estos atajos permite procesar ventas sin depender del ratón, mejorando la velocidad de atención al cliente.

---

## 🛒 Gestión del Carrito (Pantalla Principal)

| Tecla | Acción | Descripción |
| :--- | :--- | :--- |
| **Enter** | **Buscar / Cobrar** | Si el buscador tiene texto, agrega el producto. Si está vacío, abre el menú de cobro. |
| **F12** | **Ir a Cobrar** | Abre directamente el panel de pago/cobro. |
| **F2** / **➡️** | **Cambiar Unidad** | Alterna entre **Pieza (PZA)** y **Caja (CAJA)** para el producto seleccionado. |
| **F3** / **Alt + C** | **Empaque rápido** | Configura cuántas piezas contiene una caja para el producto actual. |
| **F4** / **P** / **+** | **Empacar Todo** | Convierte automáticamente las piezas sueltas en cajas completas si alcanzan el mínimo. |
| **⬆️** / **⬇️** | **Navegar** | Sube o baja entre los productos que ya están en el carrito. |
| **-** / **Supr** | **Quitar Item** | Elimina del carrito el producto que tengas seleccionado. |
| **X** | **Vaciar Todo** | Limpia completamente el carrito de compras. |

---

## 💰 Panel de Cobro (Menú de Pago)

Una vez que abres el panel de cobro (**F12**), utiliza estos atajos:

| Tecla | Acción | Notas |
| :--- | :--- | :--- |
| **F1** | **Efectivo** | Método de pago predeterminado. |
| **F2** | **Tarjeta** | Registra el pago como terminal bancaria. |
| **F3** | **Transferencia** | Registra el pago como transferencia SPEI. |
| **F4** | **Dólares (USD)** | Solo disponible si hay un tipo de cambio configurado. |
| **+** / **Enter** | **Finalizar / Abonar** | Finaliza la venta si el monto es total, o registra un abono si es pago parcial. |
| **Esc** | **Cancelar** | Cierra el panel de cobro y vuelve al carrito. |
| **0-9** | **Ingresar Monto** | Escribe directamente la cantidad recibida del cliente. |

---

## 🛠️ Funciones del Sistema

| Tecla | Acción |
| :--- | :--- |
| **F7** | **Reporte del Día** | Muestra el resumen de ventas realizadas en el turno actual. |
| **F10** | **Actualizar** | Recarga la aplicación para sincronizar precios e inventarios. |

---

> [!TIP]
> **Consejo Pro:** Mantén el cursor fuera de los cuadros de texto para que los atajos de una sola letra (como **X** o **F**) funcionen instantáneamente. Si estás escribiendo un nombre de producto, presiona **Esc** primero para "soltar" el teclado.
